# gilimall

#### 介绍

市面上常见的5种电商模式

B2B  B2C C2B C2C O2O

1、B2B模式

商家与商家建立的商业关系，如阿里巴巴。

2、B2C模式

商对客（商业零售），如天猫、京东。

3、C2B模式

消费者对企业，现有消费需求产生而后有企业按需生产。

4、C2C模式

客户之间把自己东西放到网上卖，如淘宝，闲鱼

5、O2O模式

即Online To Offine，线下商务的机会与互联网结合到一起，让互联网成为线下交易的前台。线上快速支付，线下优质服务。如饿了么，美团，京东到家



**谷粒商城是一个B2C模式的电商平台，企业销售自营商品给客户。**
